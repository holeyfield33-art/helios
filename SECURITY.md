# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 1.0.0   | Yes       |

## Reporting a vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Report vulnerabilities privately via GitHub Security Advisories:
https://github.com/holeyfield33-art/helios/security/advisories/new

Please include a description of the issue, steps to reproduce it, and any
relevant version information. You can expect an initial response within **7 days**.
